Vendor Repositories

Main vendor repositories installed via composer:

  1. Framework - a main repository that glue everything together.
  2. Backend - a backend repository for things like database, cache and mail.
  3. Frontend - a frontend repository for renderers like forms and bootstrap.
  4. Tenants - a tenants repository.
  5. Users - a users repository.
  6. Internalization - a internalization repository.
  7. Countries - a countries repository.
  8. Documentation - a documentation repository.
  9. And more...
