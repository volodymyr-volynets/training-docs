Framework Functionalities

Framework Functionalities include:

  1. Application Folders - A space for main application and other template files. More detailed is here.
  2. Vendor Folders - Open Source Numbers Repositories installed via composer.
  3. Private Folders - Private Numbers Repositories stores locally in private folder.
  4. Public HTML Folder - a folder with all HTML, JS and CSS assets.
  5. Public Sites Folder - a folder with additional source codes usually in other languages like JavaScript, Python and React or Vue.
