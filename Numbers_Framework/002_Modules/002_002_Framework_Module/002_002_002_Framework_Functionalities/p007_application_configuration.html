Application Configuration

Configuration files include:

  1. application/Config/application.ini - main configuration file.
  2. application/Config/environment.ini - additional configuration not saved to git, ideal to store credentials.
  3. application/Config/maintenance.ini - maintenance configuration file.
  4. application/Config/module.ini - module file.
  5. application/Config/registry.ini - registry file.

All configurations are in ini format.

Application Configuration example:

        
            [*,production,staging,testing,development]
            ; --- php settings ---
            php.display_startup_errors = 0
            php.display_errors = 0
            php.max_execution_time = 360
            php.memory_limit = "8192M"
            php.mbstring.func_overload = 7
            php.date.timezone = "America/Toronto" ; server timezone
            php.variables_order = "EGPCS"

            ; --- application settings ---
            application.view.mandatory = false	; whether controllers require views
            application.view.extension = "html"
            application.layout.layout = "reveal"
            application.layout.name = "Numbers Applications"
            application.layout.email = "reveal-email"
            application.layout.extension = "html"
            application.dispatch.before_controller = "\Helper\Dispatch::before"
            application.dispatch.after_controller = "\Helper\Dispatch::after"
            ...
