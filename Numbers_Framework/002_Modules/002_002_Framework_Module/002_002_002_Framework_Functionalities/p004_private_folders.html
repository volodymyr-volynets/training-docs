Private Folders

Additional private repositories installed not through composer, examples are:

  1. Job Management - a module to manage Jobs.
  2. Credentials - a module to manage passwords/credentials.
  3. FlowTracker - a analytics module.
  4. Templates - a template module.
  5. Audiences - a audience management.
  6. FakeNames - a random data generator module.

When module is ready to become open source its being moved from private to vendor folder.
