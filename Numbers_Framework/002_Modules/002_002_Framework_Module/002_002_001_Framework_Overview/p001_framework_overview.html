Framework Overview

Framework module tights all other modules together and can be broken down to following functionalities:

  1. Framework Functionalities - classes and scripts to manage framework. For details go here.
  2. ORM Classes - All classes to create and manage database objects. For details go here.
  3. Query Builder - All classes to generate dynamic SQL.
  4. Form Classes - All classes to build interactive forms.
  5. Framework Helpers - helper classes for functionalities located in other repositories for example \User and \Tenant. The names are short because they are frequently used.
  6. Type Helpers - type classes to work with a typed values. Examples are \Array2 and \DateTime.
  7. Permission Helpers - classes to with with permissions. Example is \Can.
  8. Localization Helpers - constants, localization files and classes to work with localizations and internalization.
